# Student Management System

A simple **Python** and **Tkinter** application to manage student registrations with data stored in **SQLite**. The system allows users to register students, view the student list, and visualize the gender distribution.

---

## Features

- Add new students with **Name, Email, Age, and Gender**.
- Display all registered students in a **table view**.
- Visualize the **gender distribution** with a pie chart.
- Friendly **Tkinter GUI** interface.
- Data persistence using **SQLite database**.
